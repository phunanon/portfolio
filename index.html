DOB 1997; Telford, UK

PAL Pong

Using the Arduino platform (Due/Mega 2560) I implemented the arcade classic Pong to work through PAL TV. Two potentiometers acted as basic controllers for the paddles, emulating the nostalgic feel of the first consoles! Unfortunately difficult to record due to the framerate.

OLED/PAL Raycaster

Using the Arduino platform (Mega 2560) I ported this tutorial into Arduino C, and utilised the Adafruit SSD13076 library and TVout library for display. They both used the same interface for movement as the Pong game.

Raycast First-person Shooter

For my HND Applied Computing project, I chose to create an SFML/C++ raycaster using the same core-code as in projects above, but textured.

Basic Circuit Simulator

Written in C++ for Linux terminal, this program allows you to create and step-through simple yet functional circuits. The video demonstrates my 8-bit counting machine, extendable to create a whole computer (and I have gotten further with it since!)

Mahjong

A simple implementation of the classic virtual Mahjong. Written in C++ using the SFML framework, it was a personal experiment into basic graphical applications.

RTS Game

A currently bare-bones Real Time Strategy game implemented in C++/SFML – my first isometric graphical program. My goal is to generate AI towns/villages, gigantic maps (while JIT map building is tempting, I'll probably elect to eat up RAM), and try to create a realistic economic setting.

Homebrew Pathfinding

I required a less CPU-intensive solution for a game I was planning to develop, and didn't require quickest route determination, so I took on the fun challenge of writing my own. Governed by 7 simple steps, I ported it into the RTS Game featured above. The video demonstrates a showcase of its execution. Can still be optimised further.

Brainfuck Interpreter

A black-box Brainfuck interpreter, implemented in C++. It uses a simple command register to repeat commands in a row, cached looping, and threaded display. Can be improved with better looping, and implementing optimisations as suggested by the wider community, and perhaps even rudimentary branching.
